# VPC_WebHost_Setup
AWS VPC setup with public subnet, Internet Gateway, and EC2-hosted modern website with responsive UI/UX.
VPC-WebHost-Setup

Project Overview
This project demonstrates the setup of a *VPC (Virtual Private Cloud)* on AWS, with a *public subnet, **Internet Gateway (IGW), and an **EC2 instance hosting a simple website*.  
The website is accessible via *public IP*, showcasing a modern UI/UX design.

---
 Features
- Custom *VPC* for network isolation
- *Public Subnet* with internet access
- *Internet Gateway (IGW)* attached to VPC
- *EC2 Instance* hosting a simple HTML website
- Responsive *UI/UX* design
- Fully accessible from the internet
