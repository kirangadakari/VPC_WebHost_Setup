Setup Instructions

 1. Create VPC
- Go to AWS VPC → Create VPC
- CIDR Block: 10.0.0.0/16

2. Create Public Subnet
- Subnet CIDR: 10.0.1.0/24
- Subnet Associate with VPC

 3. Create Internet Gateway (IGW)
- Attach IGW to your VPC
- Update Route Table to route 0.0.0.0/0 through IGW

 4. Launch EC2 Instance
- AMI: Amazon Linux 2
- Subnet: PublicSubnet
- Auto-assign Public IP: Enabled
- Security Group:
  - SSH (22) → Your IP
  - HTTP (80) → 0.0.0.0/0

 5. Install Apache & Host Website
--bash
- sudo apt update
- sudo apt install apache2
- sudo systemctl start apache2
- sudo systemctl enable apache2
- sudo systemctl status apache2
- cd var/www/html
- ls
- >index.html
- vi index.html
- cat index.html
